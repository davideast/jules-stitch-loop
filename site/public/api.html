API Reference

Welcome to the jules.top API. You can use our public endpoints to access developer rankings, user statistics, and repository data programmatically.

info Base URL: https://api.jules.top/v1

key Authentication

Authentication is required for some endpoints and to increase rate limits. Include your API key in the Authorization header.

BASH 
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.jules.top/v1/user/me

speed Rate Limiting

Unauthenticated

requests per hour
Authenticated

Endpoints

GET

/leaderboard

Retrieve the current global leaderboard rankings.

Query Parameters

Param Type Default Description
page integer 1 Page number for pagination.
limit integer 50 Number of items per page (max 100).

Response Example

JSON 
"data": [
  {
    "rank": 1,
    "username": "Jules",
    "avatar_url": "https://...",
    "total_ships": 8421,
    "last_active": "2023-10-27T14:30:00Z"
  },
  {
    "rank": 2,
    "username": "Alex",
    "total_ships": 7988,
    "last_active": "2023-10-27T15:45:00Z"
  }
],
"meta": {
  "total": 1450,
  "page": 1,
  "pages": 29
}
