The Protocol

jules.top tracks developer activity via Git commits and Pull Requests made with the Jules agent. It's a friendly, competitive, data-dense leaderboard for tracking developers who ship the most code.

Data Source

// Data is sourced directly from GitHub events

Scoring

Points, or 'Ships', are awarded for each Git commit and each merged Pull Request made using the Jules agent. Each action is weighted equally: 1 Commit = 1 Ship, and 1 Merged PR = 1 Ship.

Fair Play

Our protocol prioritizes data integrity. We have measures in place to monitor activity and prevent any manipulation of the leaderboard to ensure a fair and competitive environment for all developers.
